## 1. Démarrage

L’application YANGO Driver Tracker est conçue pour être utilisée sur mobile. Vous pouvez l’utiliser directement dans le navigateur ou l’installer comme une vraie application.

**Navigateurs recommandés** : Chrome, Firefox ou Edge (dernières versions)

---

## 2. Créer un compte

1. Ouvrez l’application
2. Sur l’écran de connexion, cliquez sur l’onglet **Sign Up**
3. Entrez un **nom d’utilisateur** (ex: `driver_john`)
4. Entrez un **mot de passe**
5. Confirmez votre mot de passe
6. Cliquez sur **Create Account**

> Votre compte est créé instantanément et vous êtes connecté automatiquement.

---

## 3. Se connecter

1. Entrez votre **nom d’utilisateur**
2. Entrez votre **mot de passe**
3. Cliquez sur **Login**

---

## 4. Ajouter un revenu

Il existe deux types de revenus :

- **YANGO Direct** → Courses reçues via l’application YANGO
- **Personal** → Courses privées ou personnelles

### Étapes :
1. Appuyez sur le bouton **+ Add** sur l’écran d’accueil
2. Sélectionnez **Income**
3. Choisissez le type de revenu : **YANGO Direct** ou **Personal**
4. Entrez le **montant** (en XAF)
5. (Optionnel) Ajoutez une note (ex: "Melen → Nlongkak")
6. Appuyez sur **Save Income**

> Un horodatage est automatiquement enregistré pour chaque transaction.

---

## 5. Ajouter une dépense

### Catégories disponibles :
- Carburant (Fuel)
- Entretien (Maintenance)
- Nourriture & Boissons (Food & Drinks)
- Péages & Parking (Tolls & Parking)
- Autres (Other)

### Étapes :
1. Appuyez sur **+ Add** → **Expense**
2. Choisissez une catégorie
3. Entrez le montant
4. Ajoutez une note (optionnel)
5. Appuyez sur **Save Expense**

---

## 6. Consulter les rapports

### Rapport quotidien
1. Allez dans l’onglet **Reports** (navigation en bas)
2. Sélectionnez une date
3. Consultez le détail : Revenus YANGO, Revenus personnels, Dépenses et Bénéfice net

### Rapport hebdomadaire
- Affiche automatiquement les 7 derniers jours
- Montre les totaux des revenus, dépenses et bénéfice net

---

## 7. Vue sur plusieurs jours

1. Depuis l’écran d’accueil, appuyez sur **Multiple Days View**
2. Consultez vos performances sur les 14 derniers jours
3. Chaque jour affiche le bénéfice net et le nombre de transactions

---

## 8. Exporter et importer des données

### Exporter une sauvegarde
1. Appuyez sur votre **nom d’utilisateur** (en haut à droite)
2. Choisissez l’option **1**
3. Un fichier `.json` sera téléchargé

### Importer une sauvegarde
1. Appuyez sur votre nom d’utilisateur
2. Choisissez l’option **2**
3. Sélectionnez votre fichier de sauvegarde

> Utilisez cette fonctionnalité lorsque vous changez de téléphone ou pour conserver une copie de vos enregistrements.

---

## 9. Changer son mot de passe

1. Appuyez sur votre **nom d’utilisateur** (en haut à droite)
2. Entrez **3** lorsque cela vous est demandé
3. Entrez votre mot de passe actuel
4. Entrez votre nouveau mot de passe
5. Confirmez le nouveau mot de passe

---

## 10. Installer l’application (PWA)

### Sur Android (Chrome) :
1. Ouvrez l’application dans Chrome
2. Appuyez sur les **trois points** du menu
3. Sélectionnez **"Installer l’application"** ou **"Ajouter à l’écran d’accueil"**

### Sur iOS (Safari) :
1. Ouvrez dans Safari
2. Appuyez sur le bouton **Partager**
3. Sélectionnez **"Ajouter à l’écran d’accueil"**

Une fois installée, l’application fonctionne complètement hors ligne.

---

## 11. Conseils et bonnes pratiques

- Ajoutez vos transactions **quotidiennement** pour un suivi précis
- Utilisez les notes pour vous souvenir des courses importantes
- Exportez vos données **chaque semaine** en sauvegarde
- Utilisez toujours un mot de passe fort
- Actualisez l’application régulièrement en rechargeant la page
